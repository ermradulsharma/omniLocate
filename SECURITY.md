# Security Policy

## Supported Versions

Only the latest version of **OmniLocate (v2.0+)** is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v3.0.x  | :white_check_mark: |
| v2.x    | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within OmniLocate, please send an e-mail to the maintainer via [skywalkerlknw@gmail.com](mailto:skywalkerlknw@gmail.com). All security vulnerabilities will be promptly addressed. We enforce strict SSRF and internal IP filtering by default in v2.0+.
